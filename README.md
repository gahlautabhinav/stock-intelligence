# Stock Intelligence PA

Pre-market briefing system for NSE intraday trading.

- **8:00 AM IST** — Morning agent fetches macro data, analyzes, sends Telegram + logs to GitHub
- **3:45 PM IST** — Evening agent fetches closing prices, logs actuals, sends EOD recap
- **Dashboard** — https://gahlautabhinav.github.io/stock-intelligence
